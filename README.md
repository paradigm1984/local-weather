# local-weather
a weather app based on your location

## Technologies used:

- HTML5
- CSS3
- Javascript
- jQuery
- Node JS

## NPM packages used:

- express
- nodemon

## Usage

#### First, the application gets your coordinates using the navigator.geolocation and makes an AJAX call to a weather API to get the weather based on your coordinates.
